# 🤖 Tcell 5G Campaign Bot — Инструкция по запуску

## Что нужно для запуска

- Python 3.10+
- Токен бота от @BotFather
- Telegram ID администраторов

---

## Шаг 1 — Установка зависимостей

```bash
pip install -r requirements.txt
```

---

## Шаг 2 — Настройка бота

Открой файл `tcell_5g_bot.py` и заполни:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"   # Токен от @BotFather
ADMIN_IDS = [123456789]             # Твой Telegram ID (узнать можно у @userinfobot)
COVERAGE_MAP_FILE = "coverage_map.jpg"  # Имя файла с картой 5G покрытия
```

Положи файл карты покрытия (`coverage_map.jpg`) в ту же папку что и бот.

---

## Шаг 3 — Запуск

```bash
python tcell_5g_bot.py
```

---

## Команды администратора

| Команда | Описание |
|---|---|
| `/stats` | Статистика акции (кол-во участников, ожидающих) |
| `/broadcast текст` | Отправить сообщение всем участникам |

---

## Как работает бот

```
Пользователь /start
      ↓
Выбор языка (рус / тадж)
      ↓
Приветственное сообщение
      ↓
Карта покрытия 5G + инструкция
      ↓
Пользователь отправляет скриншот
      ↓
Скриншот уходит админу в Telegram
      ↓
Админ нажимает ✅ Одобрить / ❌ Отклонить
      ↓
✅ Пользователь получает номер участника (001, 002...)
❌ Пользователь получает предложение отправить новый скриншот
```

---

## Структура файлов

```
tcell_5g_bot.py      — основной код бота
requirements.txt     — зависимости
coverage_map.jpg     — карта покрытия 5G (добавить самостоятельно)
participants.json    — создаётся автоматически при первом запуске
```

## Деплой на сервер (опционально)

Для постоянной работы можно использовать `screen` или `systemd`:

```bash
# Через screen
screen -S tcell_bot
python tcell_5g_bot.py
# Ctrl+A, D — чтобы выйти из screen
```

Или создать systemd-сервис для автозапуска.
